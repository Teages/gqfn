# @gqfn/core

GQFn provides a *typed* way to write GraphQL queries with the help of TypeScript.

## Usage

Install package:

```sh
# âœ¨ Auto-detect
npx nypm i @gqfn/core

# npm
npm install @gqfn/core

# yarn
yarn add @gqfn/core

# pnpm
pnpm install @gqfn/core

# bun
bun install @gqfn/core
```

Import:

```js
import { useSchema } from '@gqfn/core'
```
